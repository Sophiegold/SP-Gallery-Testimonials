# Testimonials
## Summary
This is a SharePoint Gallery design with testimonials.

## View requirements

|Type               |Internal Name|Required|
|-------------------|-------------|:------:|
|Hyperlink          |PhotoURL     |Yes     |
|Title (Text)       |Name         |Yes     |
|Text               |Testimonial  |Yes     |
|Text               |Email        |Yes     |

*Notes:
